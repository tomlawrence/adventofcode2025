# Advent of Code 2025

Choosing TypeScript for this year's AoC. Things should hopefully be less stressful in December this year with only 12 days of challenges!

## Instructions

```
git clone https://github.com/tomlawrence/adventofcode2025.git
cd adventofcode2025
npm install
npm start

```
